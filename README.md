# 🚗 Глянец - Панель управления автомойкой

[![.NET Build](https://github.com/iamfifya/MyPanelCarWashing/actions/workflows/dotnet.yml/badge.svg)](https://github.com/iamfifya/MyPanelCarWashing/actions/workflows/dotnet.yml)

## 📝 Описание проекта

**"Глянец"** — это полноценное desktop-приложение для управления автомойкой, разработанное на WPF с использованием надежной локальной базы данных SQLite. Система автоматизирует все бизнес-процессы: от записи клиента и контроля боксов до глубокого финансового учета, ведения кассы и сложного расчета заработной платы с учетом авансов и минимальных выплат.

Приложение спроектировано в виде Single Page Application (SPA) с использованием всплывающих оверлеев и кастомных UI-элементов для обеспечения максимального удобства администратора.

## ✨ Основные функции

### 💰 Финансовый модуль и Касса (New!)
- 💵 **Контроль наличных:** расчет фактического остатка в кассовом ящике в реальном времени.
- 📉 **Транзакции:** учет приходов (размен), расходов (химия, аренда), инкассаций.
- 💸 **Авансы:** именная выдача авансов сотрудникам с автоматическим вычетом из итоговой зарплаты.
- 💎 **Чистая прибыль (ЧПКО):** автоматический расчет реальной прибыли компании за вычетом всех расходов и доплат мойщикам.

### 📅 Управление сменами и заказами
- 🔒 Строгий контроль смен (защита от закрытия при активных заказах).
- 🚘 Управление 3 моечными боксами в реальном времени.
- 🛠️ Интеллектуальный расчет стоимости услуг в зависимости от 1 из 4 категорий кузова.
- 📋 Поддержка различных статусов ("В ожидании", "Выполняется", "Выполнен", "Отменен") и методов оплаты (Наличные, Карта, Перевод, QR).
- ➕ Добавление наценок и скидок с обязательным указанием причины.

### 📋 Предварительные записи
- ⏱️ Умная проверка занятости боксов на выбранное время (защита от накладок).
- 🔄 Бесшовное преобразование предварительной записи в активный заказ при открытии смены.

### 📊 Продвинутая отчетность
- 📉 **Интервальные отчеты:** генерация финансовой сводки за любой выбранный период.
- 💳 Разделение статистики по безналу и наличным деньгам.
- 🧾 **Зарплатная ведомость:** детальный расчет ЗП (35% от выручки) с гарантированной доплатой до МРОТ за выход и вычетом взятых авансов.
- 📄 Экспорт любой аналитики в Excel (с красивым форматированием), CSV и JSON.

### 👥 CRM и Клиентская база
- 📇 Автоматическое сохранение истории визитов, расчет LTV (Total Spent) и любимых услуг клиента.
- 🚗 Автозаполнение данных (модель, номер, категория) при повторном визите.

### 👨‍💼 Управление персоналом и Услугами
- 🔐 Разграничение прав доступа (Администратор / Мойщик).
- 🛠️ Гибкая настройка прайс-листа под разные габариты автомобилей.

## 🛠️ Технологии и Архитектура

| Компонент | Технология |
|-----------|------------|
| **Платформа** | .NET Framework 4.8 |
| **База Данных** | System.Data.SQLite (Локальная реляционная БД) |
| **Интерфейс** | WPF (Кастомные контролы, MVVM-паттерны, Overlays) |
| **Экспорт в Excel** | ClosedXML |
| **Логирование** | NLog |
| **DI контейнер** | Microsoft.Extensions.DependencyInjection |
| **Работа с JSON**| Newtonsoft.Json (для бэкапов и экспорта) |

## 📋 Системные требования

- OS: Windows 7 / 8 / 10 / 11
- Runtime: .NET Framework 4.8 или выше
- Storage: ~100 MB свободного места (включая БД и логи)
