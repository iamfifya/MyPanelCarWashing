# 🚗 Глянец - Панель управления автомойкой

## 📝 Описание проекта
Приложение для управления автомойкой с разделением на 3 бокса, учетом выручки и заработной платы сотрудников (35% мойщику, 65% компании).

## ✨ Основные функции
- ✅ Управление сменами (начало/закрытие смены)
- ✅ Добавление/редактирование/удаление заказов
- ✅ Выбор мойщика из сотрудников в смене
- ✅ Выбор бокса (1, 2 или 3)
- ✅ Выбор услуг с автоматическим расчетом суммы
- ✅ Дополнительная стоимость с указанием причины
- ✅ Управление услугами (добавление, редактирование, удаление)
- ✅ Управление сотрудниками
- ✅ Отчеты по сменам и месяцам
- ✅ Экспорт отчетов в Excel, CSV, JSON
- ✅ Статистика заработка сотрудников в реальном времени

## 🛠️ Технологии
- .NET Framework 4.8
- WPF (MVVM)
- Newtonsoft.Json
- ClosedXML (экспорт в Excel)
