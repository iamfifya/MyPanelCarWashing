# 🚗 Глянец - Панель управления автомойкой

[![.NET Build](https://github.com/iamfifya/MyPanelCarWashing/actions/workflows/dotnet.yml/badge.svg)](https://github.com/iamfifya/MyPanelCarWashing/actions/workflows/dotnet.yml)

## 📝 Описание проекта

**"Глянец"** — это desktop-приложение для управления автомойкой, разработанное на WPF. Система позволяет вести учет заказов, управлять сотрудниками, контролировать выручку и автоматически рассчитывать заработную плату мойщиков (35% от выручки).

Приложение поддерживает работу с тремя моечными боксами, предварительные записи клиентов, клиентскую базу и формирование детальных отчетов.

## ✨ Основные функции

### Управление сменами
- 📅 Начало и закрытие смены с выбором сотрудников
- 🔒 Контроль невозможности закрытия смены при активных заказах или предварительных записях
- 📊 Автоматическое формирование отчета при закрытии смены

### Работа с заказами
- ➕ Добавление, редактирование и удаление заказов
- 🚘 Выбор бокса (1, 2 или 3)
- 👤 Назначение мойщика из сотрудников текущей смены
- 🛠️ Выбор услуг с автоматическим расчетом стоимости (цены зависят от категории кузова)
- 💰 Дополнительная стоимость с обязательным указанием причины
- 📋 Статусы заказов: "Выполняется", "Выполнен", "Отменен"
- 💳 Способы оплаты: Наличные, Карта, Перевод

### Предварительные записи
- 📅 Создание записей на будущее время
- ⏱️ Проверка доступности бокса на выбранное время
- 🔄 Преобразование записи в заказ при наступлении смены
- ✏️ Редактирование и удаление записей

### Управление услугами
- 🛠️ Добавление, редактирование, удаление услуг
- 💲 Гибкая система ценообразования по 4 категориям кузова:
  - Категория 1: Легковая
  - Категория 2: Универсал
  - Категория 3: Кроссовер
  - Категория 4: Внедорожник
- ⏱️ Указание длительности каждой услуги

### Управление сотрудниками
- 👥 Добавление, редактирование, удаление сотрудников
- 🔐 Разграничение прав (Администратор / Мойщик)
- 📅 График работы сотрудников
- 📊 Статистика заработка в реальном времени

### Клиентская база
- 📇 Полноценная CRM с историей заказов
- 🚗 Автоматическое заполнение данных автомобиля при выборе клиента
- 📈 Статистика клиента: количество визитов, общая сумма, средний чек
- 🔍 Поиск клиентов по ФИО, телефону, госномеру

### Отчетность
- 📊 Отчет по смене (детализация по сотрудникам и способам оплаты)
- 📅 Месячные отчеты с разбивкой по дням и сотрудникам
- 📈 Интервальные отчеты за произвольный период
- 📄 Экспорт отчетов в Excel, CSV, JSON
- 💰 Автоматический расчет заработной платы (35% мойщику, 65% компании)
